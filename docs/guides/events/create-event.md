# Create Event Workflow

## Overview
Guide for creating new events in the PHWB system.

## Prerequisites
- User has event creation permissions
- Event details planned
- Venue and artists available

## Steps
1. Navigate to Events page
2. Click "Create Event" button
3. Fill out event details
4. Set date and time
5. Assign venue
6. Add artist assignments
7. Configure event requirements
8. Submit for creation

## Required Fields
- Event title
- Date and time
- Status
- Event type

## Optional Fields
- Venue assignment
- Program association
- Artist assignments
- Schedule details
- Requirements

## Success Criteria
- Event created successfully
- Appears in events list
- All details properly saved
