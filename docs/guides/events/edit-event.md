# Edit Event Workflow

## Overview
Guide for updating existing event information and settings.

## Prerequisites
- Event must exist in system
- User has event management permissions

## Steps
1. Navigate to Events page
2. Select event from list
3. Click "Edit" button in detail view
4. Update event information
5. Modify assignments
6. Adjust scheduling
7. Save changes

## Editable Fields
- Event details
- Date and time
- Venue assignment
- Artist assignments
- Status and requirements

## Success Criteria
- Changes saved successfully
- Updated information displayed
- Event properly maintained
