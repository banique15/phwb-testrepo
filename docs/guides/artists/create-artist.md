# Create Artist Workflow

## Overview
Guide for adding new artists to the PHWB system.

## Prerequisites
- User must have appropriate permissions
- Required artist information available

## Steps
1. Access the Artists page
2. Click "Add Artist" button
3. Fill out artist profile form
4. Set artist skills and availability
5. Add contact information
6. Submit for creation

## Required Fields
- Legal first name
- Legal last name
- Contact information

## Optional Fields
- Public display name
- Skills and specializations
- Availability preferences
- Biography/notes

## Success Criteria
- Artist profile created successfully
- Artist appears in master list
- All information properly saved
