# Delete Artist Workflow

## Overview
Guide for removing artists from the PHWB system.

## Prerequisites
- Artist must exist in system
- User must have delete permissions
- Consider impact on existing events/assignments

## Steps
1. Navigate to Artists page
2. Select artist from master list
3. Click "Delete" button in detail view
4. Review deletion confirmation dialog
5. Confirm deletion

## Important Considerations
- Check for existing event assignments
- Review payroll history
- Consider archiving vs. deletion
- Backup important information if needed

## Success Criteria
- Artist removed from system
- No longer appears in master list
- Related assignments properly handled
