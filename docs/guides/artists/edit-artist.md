# Edit Artist Workflow

## Overview
Guide for updating existing artist profiles in the PHWB system.

## Prerequisites
- Artist must exist in system
- User must have appropriate permissions

## Steps
1. Navigate to Artists page
2. Select artist from master list
3. Click "Edit" button in detail view
4. Update artist information
5. Modify skills and availability as needed
6. Save changes

## Editable Fields
- Personal information
- Contact details
- Skills and specializations
- Availability status
- Biography and notes

## Validation
- Required fields must remain populated
- Contact information format validation
- Skill assignments must be valid

## Success Criteria
- Changes saved successfully
- Updated information displayed
- Artist remains in master list
