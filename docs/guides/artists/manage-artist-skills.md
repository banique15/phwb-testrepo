# Manage Artist Skills Workflow

## Overview
Guide for managing artist skills and specializations.

## Prerequisites
- Artist profile must exist
- Skills taxonomy defined in system

## Steps
1. Access artist profile
2. Navigate to skills section
3. Add/remove skill categories
4. Set proficiency levels
5. Update availability for each skill
6. Save changes

## Skill Categories
- Musical instruments
- Performance types
- Teaching specializations
- Technical abilities
- Languages

## Success Criteria
- Skills properly assigned
- Proficiency levels set
- Availability preferences recorded
