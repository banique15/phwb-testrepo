# Artist Assignment Workflow

## Overview
Guide for assigning artists to events and programs.

## Prerequisites
- Artist profile exists
- Event/program exists
- Assignment details defined

## Steps
1. Navigate to event details
2. Access artist assignment section
3. Search and select artist
4. Define role and responsibilities
5. Set compensation details
6. Confirm assignment

## Assignment Details
- Role/position
- Hours expected
- Hourly rate
- Special requirements
- Assignment status

## Success Criteria
- Artist assigned to event
- Compensation details recorded
- Assignment status tracked
