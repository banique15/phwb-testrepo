# Edit Partner Workflow

## Overview
Guide for updating existing partner information.

## Prerequisites
- Partner must exist in system
- User has appropriate permissions

## Steps
1. Navigate to Partners page
2. Select partner from list
3. Click "Edit" button
4. Update partner information
5. Modify contact details
6. Adjust partnership terms
7. Save changes

## Editable Fields
- Organization details
- Contact information
- Partnership type
- Agreement terms
- Status and notes

## Success Criteria
- Changes saved successfully
- Updated information displayed
- Partnership terms properly recorded
