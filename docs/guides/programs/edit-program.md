# Edit Program Workflow

## Overview
Guide for updating existing program information and settings.

## Prerequisites
- Program must exist in system
- User has program management permissions

## Steps
1. Navigate to Programs page
2. Select program from list
3. Click "Edit" button
4. Update program details
5. Modify geographic coverage
6. Adjust partner assignments
7. Save changes

## Editable Fields
- Program information
- Coverage areas
- Partner relationships
- Timeline and dates
- Program parameters

## Success Criteria
- Changes saved successfully
- Updated information displayed
- Program parameters properly configured
