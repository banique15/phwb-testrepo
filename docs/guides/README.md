## Getting Started

Each guide follows a consistent structure:
- **Overview**: Purpose and scope of the workflow
- **Prerequisites**: Required conditions and permissions
- **Steps**: Detailed step-by-step instructions
- **Success Criteria**: How to verify successful completion
- **Related Documentation**: Links to relevant guides

## Navigation

Guides are organized by entity type and contain cross-references to related workflows. Each guide is designed to be used independently while providing context for the broader system workflows.

## Contributing

When updating these guides:
1. Maintain the consistent structure
2. Include specific UI references
3. Add cross-references to related workflows
