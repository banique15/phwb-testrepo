# Financial Reports Workflow

## Overview
Guide for generating financial and compensation reports.

## Prerequisites
- Payroll data available
- User has financial reporting permissions
- Reporting period defined

## Steps
1. Navigate to Reports page
2. Select Financial Reports
3. Choose reporting period
4. Select financial metrics
5. Include payroll information
6. Generate financial report
7. Review for accuracy

## Financial Metrics
- Artist compensation
- Event costs
- Program budgets
- Venue expenses
- Partnership contributions

## Success Criteria
- Financial data accurately reported
- Calculations verified
- Report complies with requirements
